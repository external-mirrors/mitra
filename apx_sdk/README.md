# apx_sdk

Minimalistic ActivityPub toolkit.

Using in a Cargo Project:

```toml
[dependencies]
apx_core = { git = "https://codeberg.org/silverpill/mitra", rev = "apx-v0.1.0" }
apx_sdk = { git = "https://codeberg.org/silverpill/mitra", rev = "apx-v0.1.0" }
```
